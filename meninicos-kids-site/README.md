# MeniNicos Kids

Projeto separado para a loja de moda infantil "MeniNicos Kids".

## Sobre
Este modelo é um site de marketplace para roupas infantis, inspirado no estilo e layout do projeto Bellas Artes.

## Arquivos
- `index.html` — página principal do site.

## Como usar
1. Abra `meninicos-kids-site/index.html` no navegador.
2. Ajuste textos, cores e produtos conforme necessário.
3. Se quiser publicar, crie um repositório novo e faça o deploy a partir desta pasta ou use outro workflow separado.
